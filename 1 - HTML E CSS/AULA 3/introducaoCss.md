# INTRODUÇÃO AO CSS

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é linguagen utilizada para definir o estilo visual de documentos HTML, controlando o layout, cores, fontes, espaçamentos e responsividade.

## Por que usar CSS?

O CSS separa o conteúdo (HTML) da apresentação. Permite deixar páginas mais atrativas e organizadas, melhorando a experiência do usuário.

## Formas de aplicar CSS:

InLine: direto no elemento, suando o atributo style.
Interno: bloco <style> dentro do <head>.
Externo: arquivo .css separado (recomendado para projetos reais).
